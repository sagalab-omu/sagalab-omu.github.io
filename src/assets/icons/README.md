# Research Area Icons Directory

This directory should contain SVG icons for research areas referenced in the JSON files.

Expected icons:
- geometry.svg (Computational Geometry)
- fabrication.svg (Digital Fabrication)
- graphics.svg (Computer Graphics)
- spatial.svg (Spatial Analysis)
- ml.svg (Machine Learning)
- vr.svg (Virtual Reality)

Recommended format: SVG
Recommended size: 48x48 pixels or scalable
The page will display placeholder icons if these files are missing.
